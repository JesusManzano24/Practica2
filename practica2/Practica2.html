1. ESTRUCTURA INTERNA DE UNA PÁGINA HTML

Ejemplo: Confeccionar una página que muestre los nombres de 5 lenguajes de programación separados por un guión.

Código que resuelve el ejercicio:


<body>
    <p>Python - JavaScript - Java - C++ - PHP</p>
</body>

Resultado:

Python - JavaScript - Java - C++ - PHP

Ejemplo: Confeccionar una página que contenga un título de primer nivel <h1> y luego dos títulos de nivel <h2>. Definir un párrafo para cada título de segundo nivel.

5. ÉNFASIS <EM> <STRONG>

Ejemplo: Crear una página que contenga cuatro párrafos. En el primero enfatizar con el máximo nivel y en los otros párrafos emplear el elemento de enfatizar de menor fuerza.

Código que resuelve el ejercicio:


    <body>
        <p><strong>Este es el primer párrafo con énfasis de máximo nivel.</strong></p>
        <p><em>Este es el segundo párrafo con énfasis de menor fuerza.</em></p>
        <p><em>Este es el tercer párrafo con énfasis de menor fuerza.</em></p>
        <p><em>Este es el cuarto párrafo con énfasis de menor fuerza.</em></p>
    </body>

Resultado:

Este es el primer párrafo con énfasis de máximo nivel.

Este es el segundo párrafo con énfasis de menor fuerza.

Este es el tercer párrafo con énfasis de menor fuerza.

Este es el cuarto párrafo con énfasis de menor fuerza.

6. HIPERVÍNCULO A OTRA PÁGINA DEL MISMO SITIO <A>

Ejemplo: Confeccionar una página principal que tenga un hipervínculo a otra página secundaria. La página secundaria debe tener también un hipervínculo a la página principal.

Código que resuelve el ejercicio:

    <!-- Página principal (index.html) -->
    <!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Página Principal</title>
    </head>
    <body>
        <h1>Página Principal</h1>
        <p>Esta es la página principal. Haga clic en el enlace para ir a la página secundaria.</p>
        <a href="secundaria.html">Ir a la Página Secundaria</a>
    </body>
    </html>
    
    <!-- Página secundaria (secundaria.html) -->
    <!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Página Secundaria</title>
    </head>
    <body>
        <h1>Página Secundaria</h1>
        <p>Esta es la página secundaria. Haga clic en el enlace para volver a la página principal.</p>
        <a href="index.html">Volver a la Página Principal</a>
    </body>
    </html>

Resultado:

Página Principal

Esta es la página principal. Haga clic en el enlace para ir a la página secundaria.

Ir a la Página Secundaria
Volver a la Pagina Principal de la Practica2

Página Secundaria

Esta es la página secundaria. Haga clic en el enlace para volver a la página principal.

Volver a la Página Principal

9. ANCLAS DE PÁGINA

Ejemplo: Confeccionar una página que contenga cuatro anclas, luego definir cuatro hipervínculos que se enlacen con dichas anclas.
